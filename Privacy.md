# DarkLens Privacy Policy

**Last Updated: July 19, 2026**

DarkLens is a browser extension designed to detect and document evidence of potentially manipulative interface patterns, commonly known as dark patterns.

## Data Collection

DarkLens does not collect, sell, or share users' personal information.

DarkLens does not use tracking or analytics and does not require users to create an account.

## Webpage Content

DarkLens analyzes relevant webpage content and DOM elements to detect potential dark patterns. This processing occurs locally within the user's browser.

DarkLens does not transmit webpage content to external servers.

DarkLens V0.2 does not send webpage content to Large Language Models (LLMs) or external AI services.

## Local Storage

DarkLens may use Chrome's local storage functionality to retain extension settings, state, and detected findings so that results can be displayed within the extension's side panel.

This locally stored information is not transmitted by DarkLens to external servers.

## Permissions

DarkLens uses browser permissions solely to provide its core functionality:

* **activeTab:** To access the webpage the user chooses to analyze.
* **scripting:** To run local detection scripts that inspect relevant webpage elements for potential dark patterns.
* **storage:** To store extension state and findings locally in the browser.
* **Host permissions:** To allow DarkLens to analyze webpages across different websites when used by the user.

These permissions are not used for advertising, tracking, profiling, or selling user data.

## Third-Party Data Sharing

DarkLens does not sell, rent, or share user data or analyzed webpage content with third parties.

## Data Security

DarkLens is designed to process webpage information locally within the user's browser. No external database or DarkLens-operated server is used to store analyzed webpage content in V0.2.

## Children's Privacy

DarkLens does not knowingly collect personal information from children.

## Changes to This Privacy Policy

This Privacy Policy may be updated as DarkLens evolves. Any changes will be reflected in the publicly available version of this policy with an updated revision date.

Future versions that introduce external services, AI models, or different data-processing practices will require corresponding updates to this Privacy Policy before those features are released.

## Contact

For questions about this Privacy Policy or DarkLens's data practices, users may contact the developer through the DarkLens GitHub repository.
